# Nvim Config

## Requirements

- Install the following packages: `git`, `make`, `unzip`, `gcc`, `ripgrep`, `tar`, `curl`
- A nerd font (like Iosevka Nerd Font)
- Install [tree-sitter-cli](https://github.com/tree-sitter/tree-sitter/blob/master/crates/cli/README.md)

## Languages Supported

- C
- Lua
